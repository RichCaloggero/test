This is meant to show how one can code transcripts similar to the edX interactive transcript without using links ("a" elements).The issue for screen reader users with the current implementation is that each link is announced, which means that attempting to read through the transcript via screen reader inserts the word "link" into the speech stream constantly, which makes reading tedious.

Here we use spans and in the javascript give them all tabindex="0". I've attempted to give them a focus outline, but since I can't see the effects of CSS via screen reader, I have no idea if it's helpful. SImilarly, I've also tried to give each a cursor via CSS so that mouse users know they can click on things. All of these effects you get for free when using anchors.
